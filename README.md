# GIBSTR

> GIB STRINGE!! Generates an alphanumeric string. That's it.

## Install

`npm i -S gibstr` or `yarn add gibstr`

## Usage

```javascript
import gibstr from 'gibstr'

console.log(gibstr()) //"cpbxf3j9qz" or something

//or

console.log(gibstr(200)) //Long string
```

## Test

`npm i && npm t`
